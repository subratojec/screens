Email Verification & Registration API

This is a backend API. Use the following endpoints:

Available Endpoints:

  • POST /verify-email - Send OTP to email address
  • POST /verify-email-otp - Verify OTP code
  • POST /register - Complete user registration

API Documentation:

  • /api-docs - JSON API documentation
  • /docs - Interactive Swagger UI documentation

Usage Flow:

  1. Send POST request to /verify-email with email address
  2. Check email for OTP code
  3. Send POST request to /verify-email-otp with email and OTP
  4. Send POST request to /register with complete user data
